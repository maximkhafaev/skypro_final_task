Git

  • git init — Инициализация репозитория
  • git clone ссылка_с_GitHub — Клонирование удаленного репозитория с GitHub локально на устройство
  • git add index.html — Добавление новых локальных файлов к отслеживанию
  • git commit -m "комментарий" — Создание коммита
  • git push — Внесение изменений на удаленный репозиторий
  • git branch имя_ветки — Создание ветки
  • git checkout имя_ветки — Переключение на ветку
